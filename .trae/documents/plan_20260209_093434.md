我将为您把网站主题从 `Ananke` 切换为 `Mainroad`。

## 1. 下载与安装主题
*   **下载主题**：将 `Mainroad` 主题克隆到 `themes/mainroad` 目录。
    *   命令：`git submodule add https://github.com/vimux/mainroad.git themes/mainroad`
    *   (如果不使用 submodule，将直接 clone 或下载源码)

## 2. 迁移与重构配置 (`hugo.toml`)
`Mainroad` 主题的配置结构与 `Ananke` 有较大差异，需要重写配置文件：
*   **基础设置**：保留原有的 `baseURL`, `title`, `languageCode` 等。
*   **启用主题**：修改 `theme = 'mainroad'`。
*   **迁移菜单**：`Mainroad` 的菜单配置位置可能略有不同，需要调整 `[menu]` 部分。
*   **新增特有配置**：
    *   **侧边栏 (Sidebar)**：配置右侧边栏的小部件（搜索、最近文章、分类等）。
    *   **作者栏 (Author Box)**：设置作者信息（如您的名字、简介）。
    *   **社交链接**：配置 Twitter/Email 等联系方式。

## 3. 清理旧代码
*   **删除旧的 404 页面**：我们之前创建的 `layouts/404.html` 可能需要根据新主题的结构进行微调或删除（如果新主题自带且样式更好）。
*   **检查 `social-share.html`**：之前修改的 Ananke 特有文件将不再生效，新主题有自己的社交分享机制。

## 4. 验证与推送
*   **本地预览**：确保新主题下文章显示正常，侧边栏小部件工作正常。
*   **推送更新**：提交所有更改到 GitHub，触发 Vercel 重新部署。

**注意**：切换主题会彻底改变网站的外观和布局，原有的 Ananke 样式将不再保留。

如果您确认切换，我将开始执行。
