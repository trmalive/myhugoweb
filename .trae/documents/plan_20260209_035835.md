### 1. 修复主题丢失问题
- **现状**：本地 `themes/ananke` 目录为空，导致 Hugo 无法生成页面。
- **操作**：我将直接下载 Ananke 主题文件并放入项目中（不再使用 Git Submodule，以避免兼容性问题），确保 Vercel 能百分之百读取到主题。

### 2. 添加 Vercel 配置文件 (`vercel.json`)
- **目的**：强制告诉 Vercel "这是一个 Hugo 项目，请执行构建命令"。
- **内容**：
  ```json
  {
    "framework": "hugo",
    "buildCommand": "hugo --gc --minify",
    "outputDirectory": "public"
  }
  ```
- **效果**：这将解决“显示代码”的问题，确保用户看到的是渲染后的网页。

### 3. 推送更新
- 将修复后的主题和 `vercel.json` 推送到 GitHub，自动触发 Vercel 的新一轮部署。
